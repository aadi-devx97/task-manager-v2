# Advanced Full Stack Task Manager

A responsive full-stack task manager built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Features

- Add / Edit / Delete tasks
- Complete / Undo tasks
- Task priorities
- Search & filter system
- Responsive mobile layout
- Keyboard shortcuts
- Loading states
- Timestamp system
- Input sanitization
- REST API backend

## Tech Stack

Frontend:
- HTML
- CSS
- Vanilla JavaScript

Backend:
- Node.js
- Express.js

## Project Structure

backend/
- server.js
- data/tasks.json

frontend/
- index.html
- style.css
- js/
